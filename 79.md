答：
这个 res的返回值 就是 **`std::ranges::in_fun_result`**

in是哪里停止了

fun 放进去的



```c++
struct Sum {
  int sum = 0;
  void operator()(const int &x) { sum += x; }
};

int main() {
  std::vector<int> contain{1, 3, 4, 5, 6};

  auto res = std::ranges::for_each(contain, Sum{});

  auto f = res.fun;
  cout << f.sum << endl;

  auto i = res.in;

  if (i == contain.end()) {
    cout << "已经到达了 结尾处了...";
  }
}

```
