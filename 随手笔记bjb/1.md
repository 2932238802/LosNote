还有个问题
如何快速区分表达式和语句

```rust
fn eight_1(x: i16) -> i16 {
    x + 1 // 这是一个表达式
}
```

- 函数体 `{ x + 1 }` 是一个表达式。
- 这个表达式的值由其**最后一个表达式** `x + 1` 决定。
- 所以，函数体 `{ x + 1 }` 的值就是 `x + 1` 的计算结果（一个 `i16`）。
- 这个 `i16` 值符合函数的返回类型声明 `-> i16`。一切正常



```rust
fn eight_2(x: i16) -> i16 {
    return x + 1; // 这是一个语句
}
```

- 函数体 `{ return x + 1; }` 也是一个表达式。
- 这个表达式的值由其**最后一个部分** `return x + 1;` 决定。
- `return x + 1;` 是一个**语句 (Statement)**，因为它以分号结尾。
- 在 Rust 中，**任何语句的值都是 `()` (空元组)**。
- 所以，函数体 `{ return x + 1; }` 这个表达式本身的值是 `()`。
