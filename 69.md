  - 在go里面 不能把 nil 赋值给 int 或者 string



- Scan 的使用

```go
var a,b int
fmt.Scan(&a,&b);
```

#### `fmt.Scanln`  这个就是比上面这个换行敏感

#### `fmt.Scanf`  格式化输入 类似C

os.Open(fileName) 返回值是什么

- 第一个返回值是 文件 指针 os.FIle
- 第二个返回值是 error 错误的接口（Interface）



- 如果打开成功 值是 nil
- 如果失败 里面就是具体的错误信息
