2. mysqlslap

- 是一个诊断程序，用户模拟mysql服务器的客户端负载，并报告每个阶段的事件，就好比多个客户端正在访问服务器一样
- mysqlslap 诊断文件的时候 不能包含注释 因为不能解析注释

5. 一些查询指令

```shell
mysql> SHOW VARIABLES;    查看服务器运行时候的系统变量的值
mysql> SHOW STATUS;     运行时候的一些统计和状态指示器

mysql> show variables like 'tmpdir';
+---------------+-------+
| Variable_name | Value |
+---------------+-------+
| tmpdir        | /tmp  |
+---------------+-------+
1 row in set (0.00 sec)
```
