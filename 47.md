什么是 聚合类

- 没有用户自定义的构造函数
- 没有虚函数
- 没有私有或者保护的非静态数据成员
- 所有非静态的数据成员都是public
- 没有带默认成员初始化器的积累

```c++

struct Data {
    int id;
    double value;
    bool flag;
};
// 聚合初始化
Data d{1, 3.14, true};

```

可以省去 构造函数
