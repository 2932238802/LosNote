1.

 QRectF
如何 理解 这个的初始化方式？



- 最直接的方式 就是 上下左右的位置 告诉它

  - x y width height

  - ```c++
    QRectF(qreal x, qreal y, qreal width, qreal height)
    ```
  
- 组合对象的方式 (左上角坐标 + 尺寸)

  - ```c++
    QRectF(const QPointF &topLeft, const QSizeF &size)
    ```
  
- 或者左上角的坐标 + 右下角的坐标

  - ```c++
    QRectF(const QPointF &topLeft, const QPointF &bottomRight)
    ```
