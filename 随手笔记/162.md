string_view 要如何优雅的使用？

```c++
1. string_view 一般是只读
2. 用字符串切分 和 修剪的时候 用这个
3. 作为函数的参数
4. 它可以同时兼容 std::string、C 风格字符串（char*）和字符数组，且不会产生临时对象的开销
```



- 构造 string的方式

```c++
#include <string>
#include <string_view>
void process() {
    std::string_view sv = "hello world";
    // 方式 A：直接构造（推荐）
    std::string s1{sv}; 
    // 方式 B：使用构造函数语法
    std::string s2(sv);
    // 方式 C：如果你在函数返回或赋值时需要转换
    // std::string s3 = sv; // 错误！不支持隐式转换
    std::string s3 = static_cast<std::string>(sv); // 正确，显式转换
}
```
