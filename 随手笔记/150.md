4. [C++]

```tex
QFileInfo 和 QFile的区别是?
```

- QFile 直接作用于 文件 处理文件的内容
  - open() 打开文件
  - read() 读取文件的内容
  - write() 向文件写入内容
  - remove() 移除文件
- QFileInfo
  - 检查文件是否存在  exists()
  - 判断文件是不是普通文件  isFile()
  - 判断是否是目录  isDir()
  - 获取文件的绝对路径  absoluteFilePath()





5. [c++ Qt]

```tex
QFileDialog::getOpenFileName
如何理解？

以及除了 这个 getOpenFileName 还有什么其它的静态函数么？
```

- 它允许用户通过图形界面 选择一个文件 并返回该文件的路径

- 常见的静态函数

  - getSaveFileName 用于打开保存文件的对话框 让用户选择文件保存路径
  - getOpenFileNames 选择文件的多选版本
  - getExistingDirectory 选择一个目录

- ```c++
  static QString QFileDialog::getOpenFileName(QWidget *parent = nullptr,
                                               const QString &caption = QString(),
                                               const QString &dir = QString(),
                                               const QString &filter = QString(),
                                               QString *selectedFilter = nullptr,
                                               Options options = Options());
  ```

  - 针对过滤器

    ```tex
    显示名称 (扩展名 扩展名)
    多个过滤器用 ;; 分隔
    ```

  - 第五个参数示例

    - ```c++
      QString selected;
      QString path = QFileDialog::getOpenFileName(
          this,
          "选择文件",
          "",
          "Images (*.png *.jpg);;Text (*.txt)",
          &selected
      );
      
      qDebug() << "用户选择的过滤器:" << selected;
      ```

  - 第六个参数

    - ```c++
      QFileDialog::DontUseNativeDialog    一般选择这个 使用Qt自己的对话框
      ```

    - ```c++
      QFileDialog::ReadOnly 	不允许用户修改文件
      ```
