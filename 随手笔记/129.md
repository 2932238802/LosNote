3. [react]

useEffect
如何理解这个函数 
以及 对应的 参数设置
react?

- 在 React 中，组件的主要工作是“根据数据渲染 UI”。除此之外的事情都叫“副作用”
- useEffect 的作用就是：告诉 React 在页面渲染完成后，去执行这段副作用代码

```js
useEffect(() => {
  // 参数1: 要执行的代码 (Effect 函数)

  return () => {
    // 可选: 清理函数 (Cleanup 函数)
  };
}, [/* 参数2: 依赖数组 (Dependency Array) */]);
```

```react
useEffect(() => {
  // 如果倒计时大于 0
  if (countdown > 0) {
    // 开启一个 1 秒后的定时器
    const timer = setTimeout(() => setCountdown(countdown - 1), 1000);
    
    // 【清理函数】：
    // 在下一次 useEffect 执行前（即 countdown 变了之前），
    // 先把上一个定时器关掉。
    return () => { clearTimeout(timer) };
  }
}, [countdown]); // 【依赖项】：只要 countdown 变了，就重新运行这段逻辑
```



4. [react]

```react
<div className="min-h-screen flex flex-col items-center justify-center p-4">

这种如何理解？
```

- **`min-h-screen`** 让这个最小的高度 等于整个屏幕的高度
- **`flex`** 开启弹性布局
- **`flex-col`** 从上到下 进行排序
- justify-center 垂直居中
- items-center 水平居中
