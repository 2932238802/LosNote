3. [rust]

CORS（跨域共享）
如何理解？



- ```
  1. 浏览器发起跨域请求
  2. 对于复杂请求，先发送OPTIONS预检请求
  3. 服务器响应允许的源、方法、头等信息
  4. 浏览器检查当前请求是否符合服务器允许的范围
  5. 如果符合，发送实际请求；否则，报错
  ```

- 当一个网页从不同的域名、端口或协议请求资源时，就产生了跨域请求。浏览器出于安全考虑，默认阻止这类请求

5.  [rust]

anyhow::Result<()> 
Result 里面 不应该都是 设置两个类型么？



- // anyhow 内部的定义

  pub type Result<T> = std::result::Result<T, anyhow::Error>;

- 写 `anyhow::Result<()>` 时：

  - **成功类型** 是 `()`（即 unit，表示不返回任何数据）。
  - **错误类型** 被**固定**成了 `anyhow::Error`。

  所以，它其实依然是两个类型，只不过 `anyhow` 帮你把第二个类型（错误类型）给“包办”了

- anyhow
  - 自动转换
  - 只要一个错误实现了 `std::error::Error` 特性，你就可以在函数里直接用 `?`，它会自动把各种乱七八糟的错误转成 `anyhow::Error`
