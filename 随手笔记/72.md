1. [mysql]

mysql
里面 set GLOBEL 和 set session 来控制的变量是局部变量吗？



| 变量类型         | 作用域          | 生命周期       | 设置命令      | 配置文件修改       | 示例                                            |
| :--------------- | :-------------- | :------------- | :------------ | :----------------- | :---------------------------------------------- |
| **全局系统变量** | 整个MySQL服务器 | 重启前有效     | `SET GLOBAL`  | ✅ **可以**（推荐） | `SET GLOBAL max_connections = 1000;`            |
| **会话系统变量** | 当前连接        | 当前连接期间   | `SET SESSION` | ✅ **可以**         | `SET SESSION sql_mode = 'STRICT_TRANS_TABLES';` |
| **用户变量**     | 当前连接        | 当前连接期间   | `SET @var`    | ❌ **不可以**       | `SET @counter = 0;`                             |
| **局部变量**     | 存储过程/函数内 | 程序块执行期间 | `DECLARE`     | ❌ **不可以**       | `DECLARE x INT DEFAULT 0;`                      |
