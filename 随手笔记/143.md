- ```c++
  QUrl url("http://api.game.com");
  url.setPath("/v1/login"); 
  // 最终 url 变成: http://api.game.com/v1/login
  
  url.setPath("register"); // 注意没写斜杠
  ```

- #### 核心初始化工作

  当你调用 `QNetworkRequest(url)` 时，Qt 内部完成了以下初始化：

  - **绑定目标地址**：将上面构建好的 `QUrl` 存入请求对象中。这是请求最核心的信息：去哪里？
  - **初始化 Header 容器**：内部创建一个空的 Map，用来存放 HTTP 头部信息（如 `User-Agent`, `Content-Type` 等）。
  - **默认配置设置**：
    - **优先级**：默认设置为 `NormalPriority`。
    - **缓存策略**：默认通常是 `UseNetworkRequestCachePolicy`（遵循服务器缓存指令）。
    - **重定向追踪**：在 Qt 6 中，默认会自动处理简单的重定向
