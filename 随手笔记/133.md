如何 在 unordered_map 预先开辟n个空间 都为0？

- ```c++
  std::unordered_map<int, int> myMap;
  myMap.reserve(n); // 预留足够的桶空间，防止插入 n 个元素时发生扩容
  ```

- ```c++
  std::unordered_map<int, int> myMap;
  myMap.reserve(n); // 先预留空间提高效率
  
  for (int i = 1; i <= n; ++i) {
      myMap[i] = 0; // 显式初始化
  }
  ```

- ```c++
  std::unordered_map<int, int> myMap;
  myMap.reserve(100); // 性能优化：预留100个位置
  
  // 此时 map 是空的
  // 但当你访问任意一个 key 时：
  int value = myMap[10]; 
  
  // 此时 map 里会自动多出一条数据：{10: 0}
  // 因为 10 原本不存在，C++ 帮你创建了它并默认赋值为 0
  
  ```
