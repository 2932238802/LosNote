定义宏

```rust
macro_rules! say_hello {
    // 匹配规则：当宏被调用且括号内为空时
    () => {
        // 展开成这行代码
        println!("Hello, World!");
    };
}

fn main() {
    // 调用宏
    say_hello!();
}
```

3. [rust]

```rust
impl MyStruct {
    // 我们把它命名为 create_with_theme
    fn create_with_theme(theme: &str) -> Self { // 注意这里返回 Self
        println!("----- 创建 MyStruct -----");
        println!("主题是: {}", theme);
        
        // 这是真正创建结构体实例的语法
        Self { 
            learning_theme: theme.to_string(),
        }
    }
}
```

这里的

Self { 
            learning_theme: theme.to_string(),
        }

可以用分号嘛？ 为什么要用逗号？
