3.  C++ 17

[[fallthrough]]		

如何理解？

就是 专门用来告诉 编译器和代码阅读者
switch 语句里面 之所以 不写 break 是故意为之的



```c++
int main() {

  int i = 1;

  switch (i) {
  case 1: {
    cout << 1;
    [[fallthrough]]; // 表示 就是 故意为之的
  }
  case 2: {
    cout << 2;
    break;
  }
  default: {
    cout << 3;
  }
  }

  return 0;
}
```

打印1和2之后 就退出
