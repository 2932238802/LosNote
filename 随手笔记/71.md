1. [common]

Ring模型

- Ring 0 内核态: 拥有最高权限 可以执行所有 CPU 指令  可以直接访问硬件（如网卡 磁盘）
  - 操作系统内核就运行在这个级别
- Ring 3 用户态: 权限受限 只能执行一部分指令 不能直接访问硬件
  - 我们写的应用程序都运行在这个级别

- 系统调用或者中断 实现 从Ring3 到 Ring 0
