5. json_encode 的翻译逻辑

```php
$user = [
    "name" => "Neo",     // 关联数组 -> 翻译成 JSON 对象 {}
    "role" => 1,          // 数字 -> 翻译成 JSON 数字
    "tags" => ["A", "B"] // 索引数组 -> 翻译成 JSON 数组 []
];

$jsonString = json_encode($user);

echo $jsonString;
// 结果：{"name":"Neo","role":1,"tags":["A","B"]}

```
