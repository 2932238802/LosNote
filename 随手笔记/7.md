2. Qt 绘制三角形

```c++
QPolygon polygon;
int x = ui->slienceBtn->x();
int y = ui->slienceBtn->y();
int height = ui->slienceBtn->height();
int width = ui->slienceBtn->width();
QPoint global = ui->slienceBtn->mapToGlobal(QPoint(x,y));
QPoint point1(global.x() + 3,global.y() + height + 3);
QPoint point2(global.x() + width - 3,global.y() + height + 3);
QPoint point3(global.x() + width/2 ,global.y() + height + 30);
```

如何绘制呢
```c++
int x = ui->slienceBtn->x();
    int y = ui->slienceBtn->y();
    int height = ui->slienceBtn->height();
    int width = ui->slienceBtn->width();
    QPoint point1(x + 3, y + height + 3);
    QPoint point2(x + width - 3, y + height + 3);
    QPoint point3(x + width / 2, y + height + 30);
    QPolygon polygon;
    polygon << point1 << point2 << point3;
    painter.setPen(Qt::NoPen);
    painter.setBrush(QColor(50, 50, 50));
    painter.drawPolygon(polygon);
```
