设置最大的匹配个数 为 [1,50000] 如何在 QLineEdit 写？



- ```c++
  ui->maxMatchNumLE->setValidator(new QIntValidator(1, 50000, this));
  
  // 2. Connect the signal
  connect(ui->maxMatchNumLE, &QLineEdit::textChanged, this, [=](const QString &text) {
      // Convert
      bool ok;
      int val = text.toInt(&ok);
  
      // Logic to handle empty or intermediate states
      if (!ok) return; // Or handle empty
  
      // Logic to clamp strictly (Validator does visuals, this does data integrity)
      if (val < 1) val = 1; 
      if (val > 50000) val = 50000;
  
      emit _imageInfoParamChanged(HS_curTemplateName, IMAGE_INFO_PARAM_TYPE::MAXMATCHNUM, val);
  });
  ```

```rust
resize_exact
这个是成员函数么？

FilterType::Triangle
这个也是 image库 自带的么？
```

- 是的 这两个都是 image库 自带的

- `resize_exact` 是方法
  - 强制图片 缩放到 指定的宽度和高度
  - 返回一个 处理之后的 图片结果

- **`FilterType::Nearest`**（最邻近法）：最快，但缩放后会有很重的锯齿（马赛克感）。

- **`FilterType::Triangle`**（三角形插值/双线性）：速度和质量的平衡点。**在做 AI 识别（如 MNIST）时，通常首选这个。**
- **`FilterType::CatmullRom`**：质量更高，更锐利，但计算量稍大。
- **`FilterType::Lanczos3`**：质量最高，适合高质量图像缩放



5. [rust]

```rust
fn main() -> Result<(), Box<dyn Error>>
{
    let img = image::open("./test.bmp")?;

    let (x,y)  = img.dimensions();
    println!("x: {}",x);
    println!("y: {}",y);

    let scaled_image = img.resize_exact(50, 50, image::imageops::FilterType::Triangle);

    let gray_img = scaled_image.to_luma8();

    gray_img.save("test2.png").unwrap();
    Ok(())
}
```



- 这样就能实现 图片的缩放
