注意设计的盲点

```rust
#[tokio::main]
async fn main() -> Result<(), Box<dyn Error>> {
    let (read_half, mut write_half) = socket.into_split();

    // 任务1：后台听消息
    tokio::spawn(async move {
        // ... 读取服务器代码 ...
    });

    // 任务2：后台读键盘（假设我们也 spawn 了）
    tokio::spawn(async move {
        // ... 读取键盘代码 ...
    });

    // ---> 代码运行到这里，main 函数结束了！
    // ---> 整个进程销毁，任务1和任务2还没来得及跑就被杀死了。
    Ok(())
}
```

如果本地已经有了 一个 .git 我要如何 和 远端仓库 进行联系呢？
详细讲解一下

- 第一步 就是 先 git add git commit 把东西先放在仓库里面
- 第二部 就是 改主要的分支 git branch -M main
- 第三步 就是 git remote add origin <远端仓库链接>
- 第四步 就是 git push -u origin main 就可以推送过去了
