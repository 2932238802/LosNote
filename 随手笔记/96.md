1. [rust]

fn main()
{
    let arr_1 = [1,2,3,4,5,6];
    let number_1 = arr_1[1];
    let number_1_copy = arr_1[1];

    print!("{}",number_1);
    print!("{}",number_1_copy);
}

这个为什么 没有拿走 索引为1的所有权？



- Rust 中的基本类型（如整数、浮点数、布尔值、字符等）都实现了 `Copy` trait。这意味着它们的大小是固定的，并且存储在栈（Stack）上，复制它们的成本非常低

2. [rust]

```rust
fn main()
{
    let mut number =  1;
    let mut sum = 0 ;
    let number_1_to_100 = loop{
        sum = sum + number;
        number = number + 1; 
        if number == 100 {
            break sum
        }
    };

    print!("{}",number_1_to_100);
}
```

算出来是 4950 也就是 1~100 的和

3. [rust]

```rust
fn main(){
    let mut sum = 0;
    let mut number = 1;
    while number != 100{
        sum += number;
        number = number + 1;
    }
    print!("{}",sum);
}
```

- 这个也是4950

5. [rust] 倒计时的逻辑

```rust

fn main(){
    for i in (1..4).rev() {
        println!("打印{}",i);
    }
}
```
